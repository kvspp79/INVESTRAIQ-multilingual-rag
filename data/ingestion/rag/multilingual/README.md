This module enables Indic language support.
It handles multilingual embeddings and translation so users can
ask and receive answers in Indian languages.
## Project Setup
1. Clone the repository from GitHub.
2. Install Python 3.9 or above.
3. Install required libraries (future scope).

## Workflow
1. Collect startup funding data from reports, PDFs, and web sources.
2. Ingest and preprocess data using the ingestion module.
3. Store embeddings in a vector database.
4. Use RAG to retrieve relevant documents.
5. Generate multilingual, source-grounded responses.
